前言：本目录包含各种类型的 Galgame ，都是鄙人亲自玩过一遍写上的，并且根据个人主观评价分了等级的，请保持自己的思考，文明游戏，我只是个臭打游戏的。声明一下，可以标注来源的我会标注来源，标注不了的，大家各凭本事找资源吧。

# 五星

无论是剧情、人物塑造、CG、游戏设计都超级好，甚至部分游戏对鄙人价值观影响较大（积极正面的）

  • 《ATRI My Dear Moments》：steam - 女主设定仿生机器人，故事背景海平面急剧上升后的生活，主要围绕亚托莉与夏生共同度过的 44 天，算是 good end
  • 《他人世界末》：steam - 国内开发的校园爱情 galgame，代入感特别强，不是龙傲天爽文，也不是梦中情节，剧情很真实。
  • 《鸢之歌》：steam - 国内游戏，末世生存，背景设定很大，剧情很值得推敲，游戏制作也很精良
  • 《电车 X 同级生》： Local Love 三部曲第二部，其实三部剧情都挺离谱的，但是女主在田川鳰太戳我 xp 了
  • 《可爱的圣诞礼物：泡沫冬景》 ：steam - 猫猫社 Nekoday 开发，公司成立在四川成都，四川人打 call！要钱，整个游戏就是纯剧情，无选择。男主在 1988 年从中国到日本谋生展开的故事，跟爱情没关系，出场的人物都很个性鲜明
  • 《IRIS》： TapTap 免费游戏，但是 21 年暂停下载了，大家想玩就各凭本事了吧，故事关于战斗机器人 都是女性 在战争后的生活，男主类似于星探，把这些女孩组成一个明星团，当然是纯爱，还面临生死考验，颜值、剧情都 ok

# 四星

没什么毛病，细节处理也很到位，但是还没有那么让人印象深刻，不失为一部佳作

  • 《高考恋爱 100 天》：steam - 国内开发，初中看到视频里有人玩，算是难忘回忆了吧
  • 《零下记忆》 ：TapTap 游戏，国内开发，推理过剧情的游戏，略带惊悚设定和情节，但都不是那种值得品味的剧情，背景独特，游戏制作、人物画像等方面都很好，就是要钱
  • 《青箱（测试服）》 ：TapTap 游戏，国内开发，我没玩完，因为要钱，游戏制作上也很好，场景转换流畅，有些独特的体验

# 三星

吃饭无聊的时候可以当电视看看

  • 《密语》：兄妹禁忌，父母双亡，男主被父系族人收养，女主被母系族人收养，在不同城市，后来男主出差到了妹妹所在的城市，此时妹妹还是高中生，懂的都懂，good end 走向
  • 《同居 X 后辈》：Local Love 三部曲第一部，怎么说呢，正常的小日子纯爱剧情
  • 《星空列车与白的旅行》：也许它很值得玩，但是我实在提不起兴趣，一个关于救赎的故事，但是节奏太慢了，客观来说还是很推荐的
  • 《露西她所期望的一切》：steam，在 TapTap 中也有，我玩过一个 Bad End，就是露西被男主老爸烧了，就没心思玩了

# 二星

对我而言类似于肥皂剧，有点荒谬牵强的感觉

  • 《神社 X 前辈》：Local Love 三部曲第三部，连神社的神灵都出来了，没什么看头

# 一星

屑作，里面要么就是工业糖精，要么就是一些恶趣味，自以为黑暗的游戏（我这个评价太主观了，没有任何参考价值，最好的方式就是你自己去玩一把）

  • 《濒死轮回的卿于馆中萌生的憎恶》：一些无聊的虐待场景，被称为第十三神器，个人感觉设定和剧情都不怎么行
